Hvordan lage en nettside på GitHub

1. Lag en bruker på github.com

2. Lag en Repository, ved å klikke på "New" oppe til venstre

3. Velg et navn til Repository, gjør den public og klikk "Add a read me file"

4. Legg til en ny fil og kall denne index.html

5. Skriv inn koden til nettsiden din i index.html

6. Trykk inn på Options og finn GitHub Pages. Der velger du Branchen "Master" og mappen /root og trykker save

7. Linken til nettsiden vil komme øverst under GitHub Pages
